无符号运算是没有溢出的
有符号运算是有溢出的

```cpp
// 通过这样的方式判断有符号运算的溢出是不可靠的
if (a + b < 0) 
    complain();

// 不同的编译器实现该程序的方式可能是不同的。有编译可以通过判断内部寄存器正负号的标志是否为 “负” 来判断。这样两个整型数据相加溢出的判断永远不会成功
```


```cpp
// 这是两种正确的判断方式
if ((unsigned)a + (unsigned)b > INT_MAX) 
    complain();

// 从运算符的优先级可以知道，先计算 INT_MAX - b，再进行比较
if (a > INT_MAX - b) 
    complain();
```
